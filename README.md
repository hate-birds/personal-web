Приложение создано с помощью [Next.js](https://nextjs.org/)

## Getting Started

Для запуска development сервера:

```bash
cd .\website\
```

Затем

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере для того, чтобы увидеть результат.
